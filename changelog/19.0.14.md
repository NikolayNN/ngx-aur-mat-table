## ngx-aur-mat-table v19.0.14

**Исправления ошибок**

- **SelectionProviderDummy.isEnabled** — было `true` вместо `false`, что приводило к созданию лишних DOM-элементов при отключённом selection
- **Конфликт поискового фильтра и программных фильтров** — `applySearchFilter` и `applyFilter` затирали друг друга. Теперь оба механизма объединены в едином `filterPredicate`
- **Авто-генерация колонок** — `removeWrongKeysFromDisplayColumns` случайно выставлял `_customDisplayColumnsEnabled = true`, блокируя обновление колонок при изменении `columnsCfg`
- **updateColumnOffsets с extra header rows** — `querySelectorAll('th')` захватывал `<th>` из всех header rows, создавая записи с `key: undefined`. Теперь берётся только `_displayColumns.length` элементов
- **Drag preview debug log** — убран `console.log` из `DragPreviewManager.showPreview`, который выполнялся при каждом начале перетаскивания
- **endDrag при null от afterDropFn** — если `afterDropFn` возвращал null, `dropEvent` и `startDragEvent` не очищались и протекали в следующий цикл drag-drop
- **Total row со значением 0** — `|| ''` заменён на `?? ''`, чтобы значение `0` корректно отображалось в footer
- **ContainsStringIgnoreCase с null value** — фильтр с пустым значением искал литерал `"undefined"` в данных вместо пропуска всех строк

**Производительность**

- Убран `ngAfterViewChecked` — DOM-запрос `querySelectorAll('th')` выполнялся на каждом цикле change detection. Теперь обновление offset-ов происходит только через `ResizeObserver`

**Прочее**

- Удалён мёртвый код `hoverTimer`
- `console.log` заменён на `console.warn` для дубликатов drag-drop маппингов
- Удалён неиспользуемый import `OnInit` из `ColumnViewComponent`
- Безопасный `removePreview` через `parentNode?.removeChild` вместо `document.body.removeChild`

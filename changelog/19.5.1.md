# 19.5.1

Correctness patch: three bug fixes, no API changes.

## Fixed

- **Search no longer produces false matches.** The default search predicate
  concatenated all internal fields of the row wrapper, including the row index
  and the source object placeholder (`[object Object]`): searching `1` matched
  rows by their hidden index, searching `object` matched everything. The search
  now scans the configured column values only (same all-columns semantics,
  without the internal fields).
- **`indexCfg.name` renders in the index column header.** The configured name
  was silently dropped (projected into a component without a content outlet),
  so the index column header always appeared empty.
- **Icon wrapper (`icon.wrapper`) gets its circle styling.** The 28×28 rounded
  background was defined in a style scope it could never apply from; wrapped
  icons rendered as an unstyled color patch. The rule now lives with the
  element it styles.

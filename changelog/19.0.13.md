## ngx-aur-mat-table v19.0.13

**Новое: настройка ширины колонок**

Добавлена возможность задавать `width`, `minWidth` и `maxWidth` для колонок через новое свойство `size` в `ColumnConfig`.

```typescript
{
  name: 'Name',
  key: 'name',
  valueConverter: v => v.name,
  size: { minWidth: '150px', maxWidth: '300px' }
}
```

Поддерживаются любые CSS-единицы (`px`, `%`, `em` и т.д.). Свойство опциональное — без `size` колонки работают как раньше.

**Новый тип:** `ColumnSize` — доступен для импорта из `ngx-aur-mat-table`.

Клик заменяет [tableData] новым массивом новых объектов (как ответ REST): статус-иконка / цвет / tooltip / код действия пересчитываются (фикс #1/#3). У несовершеннолетних действие «удалить» и пункт меню «В архив» становятся disabled — их цвет (красный / фиолетовый) гаснет в нейтральный disabled-цвет (фикс #2).
